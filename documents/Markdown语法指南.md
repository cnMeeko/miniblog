# Markdown 语法指南

本文档介绍了 MiniBlog 支持的 Markdown 语法。

## 标题

使用 `#` 符号创建标题，支持的级别从 1 到 6：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本格式

**粗体文本** 使用两个星号：`**文本**`

*斜体文本* 使用一个星号：`*文本*`

`行内代码` 使用反引号：`` `代码` ``

## 代码块

使用三个反引号创建代码块：

```php
<?php
echo "Hello, World!";
?>
```

## 链接

创建链接的语法：

```markdown
[链接文本](https://example.com)
```

示例：[访问 Google](https://www.google.com)

## 列表

### 无序列表

- 项目 1
- 项目 2
- 项目 3

### 有序列表

1. 第一项
2. 第二项
3. 第三项

## 引用

使用 `>` 符号创建引用：

> 这是一个引用块。
> 可以包含多行内容。

## 分隔线

使用三个或更多的连字符、星号或下划线：

---

## 图片

图片会自动与文章关联：

![示例图片](image.php?title=欢迎使用MiniBlog&ext=jpg)

## 转义字符

如果需要显示 Markdown 特殊字符，使用反斜杠转义：

\*不是斜体\*
\**不是粗体\**

## 最佳实践

1. 使用标题组织内容结构
2. 适当使用列表提高可读性
3. 代码块使用正确的语言标识
4. 图片文件名与文章标题保持一致
5. 定期备份重要文章

# 欢迎使用 MiniBlog

这是一个简单而强大的无数据库博客系统。所有文章都以 Markdown 格式存储在 `documents/` 目录中。

## 主要特性

- **无数据库**: 所有内容存储为 Markdown 文件
- **安全**: 内置防注入、防XSS、防遍历等安全防护
- **管理后台**: 完整的文章增删改查功能
- **导入导出**: 支持单篇文章的导入导出
- **备份恢复**: 支持全站备份和恢复
- **图片支持**: 支持上传和显示文章图片
- **搜索功能**: 支持全文搜索
- **响应式设计**: 适配各种设备

## 如何使用

### 创建文章

1. 登录管理后台
2. 点击"新建文章"
3. 输入标题和内容（支持 Markdown 语法）
4. 点击"保存文章"

### Markdown 语法示例

**粗体文本**

*斜体文本*

`行内代码`

```
代码块
```

[链接文本](https://example.com)

### 列表示例

- 无序列表项 1
- 无序列表项 2
- 无序列表项 3

1. 有序列表项 1
2. 有序列表项 2
3. 有序列表项 3

## 技术栈

- PHP 8.0+
- Apache + mod_rewrite
- 纯文件存储（无数据库）

## 安全特性

- CSRF 防护
- 登录频率限制
- 会话超时（30分钟）
- 输入过滤和验证
- 文件上传验证
- 路径遍历防护
- XSS 防护
- 安全日志记录

开始使用吧！
